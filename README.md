# acai-ts-docs

Documentation repository for the acai-ts project

## Development

### Setup

```bash
# Install Python dependencies
pipenv install

# Run local development server
pipenv run serve
```

### Build

```bash
# Build production site
pipenv run build
```

The built site will be in the `docs/` directory.
